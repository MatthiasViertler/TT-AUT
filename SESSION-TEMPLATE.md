# Session Starter Template

Copy-paste this at the start of every new Claude conversation for this project.
Replace the last two lines with your actual task and relevant code.

---

```
[paste contents of CLAUDE.md here]

Today's task: <one clear sentence, e.g. "Add SAXO bank parser">
Relevant code:
<paste only the specific function(s) you need help with, not entire files>
```

---

## Tips for efficient sessions

- Paste CLAUDE.md at the start — always. Claude has no memory between sessions.
- One task per session where possible. Start fresh for unrelated topics.
- Paste only the relevant function, not the whole file.
- Include exact error messages when debugging (you've been doing this well).
- End sessions that changed something significant with:
  "Update CLAUDE.md to reflect: [what changed]"
- If a section of CLAUDE.md hasn't been relevant in 3+ sessions, move it to CLAUDE-archive.md.

## What NOT to do

- Don't ask Claude to "remember for next time" — it won't.
- Don't continue a long debug session for an unrelated question.
- Don't paste entire files when one function is enough.
- Don't let CLAUDE.md go stale — outdated context is worse than no context.
